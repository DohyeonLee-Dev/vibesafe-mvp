# VibeSafe

> AI-powered security scanner for vibe-coded projects — catch vulnerabilities before they ship

## Why VibeSafe?

The rise of "vibe coding" (AI-generated code via Cursor, Copilot, Claude, etc.) means more code is shipped without security review. VibeSafe catches the most common security mistakes: hardcoded secrets, SQL injection, XSS, disabled SSL, and more.

## Target Users

Solo developers, indie hackers, and small teams shipping AI-generated code fast.

## Quick Start

```bash
pip install -r requirements.txt
python -m vibesafe.cli scan /path/to/your/project
```

## Usage

### Scan a project
```bash
# Table output (default)
vibesafe scan ./my-project

# JSON output
vibesafe scan ./my-project --format json --output report.json

# HTML report
vibesafe scan ./my-project --format html --output report.html

# Filter by severity
vibesafe scan ./my-project --severity high

# Ignore specific rules
vibesafe scan ./my-project --ignore VS-QLT-001 --ignore VS-QLT-002
```

### Web dashboard
```bash
vibesafe dashboard ./my-project --port 8000
# Open http://localhost:8000
```

### List all rules
```bash
vibesafe rules
```
